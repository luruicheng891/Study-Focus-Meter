# 一、项目定位

本项目是一个：

```text
基于 STM32H7 的多模态边缘AI学习状态感知系统
```

系统融合：

- AI视觉分析
- 姿态检测
- 环境监测
- 无线低功耗IoT节点
- FreeRTOS实时系统
- LVGL图形界面

用于：

```text
实时分析用户学习状态与专注度
```

并提供：

- 坐姿提醒
- 环境优化建议
- 疲劳检测
- 学习时长统计

------

# 二、整体系统架构

整体采用：

```text
感知层
↓
无线边缘节点层
↓
STM32H7边缘AI主控层
↓
多模态融合层
↓
LVGL交互层
```

属于典型：

```text
IoT + Edge AI + Embedded System
```

架构。

# 3️⃣ 无线椅子节点（IoT边缘节点）

这是项目的重要创新点。

------

## 从机MCU

Nordic nRF52832

------

## 传感器

| 模块       | 功能         |
| ---------- | ------------ |
| 压力传感器 | 检测是否坐下 |
| MPU6050    | 姿态检测     |

------

## 数据采集方式

### 压力传感器

- ADC采集
- 阈值判断

### MPU6050

- I2C读取
- Motion Interrupt中断唤醒

------

## BLE通信流程

```text
压力/姿态变化
↓
唤醒nRF52832
↓
采集数据
↓
BLE发送
↓
STM32H7接收
↓
重新Sleep
```

------

## 低功耗设计

系统采用：

```text
Sleep
↓
中断唤醒
↓
采集
↓
BLE发送
↓
重新Sleep
```

低功耗IoT架构。

## 设计原则

禁止：


- MCU资源有限

- 提高稳定性
