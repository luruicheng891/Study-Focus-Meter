====================================================================
  ILI9341 LCD Driver - STM32H723ZGT6 HAL SPI4 Port
====================================================================

移植自 QDtech STM32F407 硬件SPI驱动，适配 STM32H723ZGT6 + HAL库。

====================================================================
引脚分配
====================================================================

  LCD模块          STM32H723         说明
  ─────────────────────────────────────────────────
  SDI(MOSI)   ->   PE6              SPI4_MOSI (AF5)
  SDO(MISO)   ->   PE5              SPI4_MISO (AF5)
  SCK         ->   PE2              SPI4_SCK  (AF5)
  CS          ->   PD0              GPIO 输出
  DC/RS       ->   PD1              GPIO 输出
  RST         ->   PD2              GPIO 输出
  LED         ->   PD3              GPIO 输出 (背光)
  VCC         ->   3.3V
  GND         ->   GND

====================================================================
文件说明
====================================================================

  lcd.h       - LCD驱动头文件（引脚宏定义、颜色定义、函数声明）
  lcd.c       - LCD驱动实现（ILI9341初始化序列、绘图函数）
  lcd_spi.h   - SPI4初始化头文件
  lcd_spi.c   - SPI4硬件初始化（GPIO复用、SPI外设配置）

====================================================================
使用方法
====================================================================

1. 在 STM32CubeMX 中创建 H723 工程，不需要在 CubeMX 中配置 SPI4，
   驱动内部会自行初始化 SPI4 和相关 GPIO。

2. 将这4个文件加入工程，确保 include 路径包含本目录。

3. 在 main.c 中调用：

   #include "lcd.h"

   int main(void)
   {
       HAL_Init();
       SystemClock_Config();  // 配置好系统时钟

       LCD_Init();            // 初始化LCD（内部会初始化SPI4+GPIO）

       LCD_Clear(RED);        // 清屏为红色
       
       POINT_COLOR = WHITE;
       LCD_DrawLine(0, 0, 239, 319);  // 画一条白色对角线

       while (1) {}
   }

====================================================================
时钟说明
====================================================================

  STM32H723 默认系统时钟 275MHz，APB2 = 137.5MHz。
  SPI4 分频系数设为 4，SPI时钟约 34MHz。
  ILI9341 写操作最高支持约 40MHz，读操作最高约 16MHz。
  
  如果你的时钟树不同，请修改 lcd_spi.c 中的：
    hlcd_spi.Init.BaudRatePrescaler = SPI_BAUDRATEPRESCALER_4;
  
  可选值: 2, 4, 8, 16, 32, 64, 128, 256

====================================================================
注意事项
====================================================================

  - 如果不需要读取LCD数据，MISO(PE5)可以不接。
  - 背光LED引脚如果需要PWM调光，可改为TIM输出，此处默认GPIO直接控制。
  - 触摸屏部分未移植，如需要请另行添加。
  - 确保工程中已包含 HAL 库的 stm32h7xx_hal_spi.c 和 stm32h7xx_hal_gpio.c。
